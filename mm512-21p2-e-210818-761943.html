Considere las subvariedades Riemannianas de $\RR\pow3$ dadas por: \begin{align*} M_1:&x\pow2 + y\pow2 - z\pow2 - 1 = 0 \\ M_2:&x\pow2 + y -1 = 0 \end{align*}

Haga lo siguiente (20% c/u):

problema 1 ) Encuentre el tensor métrico de $M_1$ para una carta o una parametrización

problema 2 ) Calcular los símbolos de Cristoffel para una carta o parametrización de $M_1$.

problema 3 ) Dada en $M_1$ la curva $$ \alpha:t\mapsto (1,t,t) $$

  1. Encuentre $\alpha'$ en $TM$
  2. Determine si la curva es geodésica.

problema 4 ) En la variedad $M_2$, transporte el vector $V_p$, $v=(1,0,0)$, $p=(0,1,0)$, al punto $q=(1,0,0)$. Naturalmente, use coordenadas o parametrización adecuadas y las ecuaciones de transporte paralelo.

problema 5 ) Calcule ya sea $R_{1,1,1}^1$ o $R_{1,1,1,1}$ para la variedad $M_2$.

Observaciones:

  • Para no hacer más trabajo que el necesario, escoja sus cartas o parametrizaciones de acuerdo a todos los incisos en que vaya a usarlas, por ejemplo, vea el 1,2 y 3.
  • Recuerde que una parametrización tiene sentido cuando se restringe su dominio para dar una carta, pero en la práctica puede usarse de una forma más o menos libre.
  • Los símbolos de Cristoffel puede calcularlos ya sea con la fórmula usando el tensor métrico o usando la definición.
  • Los dibujos pueden resultarle buena guía para su trabajo.
  • Algo obvio: para ver si un vector está en un plano P con normal n, basta ver que es ortogonal a n.
